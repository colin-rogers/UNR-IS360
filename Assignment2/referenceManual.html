HTML <a> Tag & Hyperlinks

Take me there:

Abstract

In HTML, the <a> is used to create a hyperlink that is used to navigate pages within our site, or to links on the web.

These links come in two varieties:

  • Absolute links
  • Relative links

The <a> can also be used to create bookmarks.

  • Links to a position on the same page.

As well as configure where the new page appears.

  • In a new browser tab, or replace the existing page.

Details

Definition and Usage

The <a> tag defines a hyperlink, which is used to link from one page to another.

The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

By default, links will appear as follows in all browsers:

  • An unvisted link is underlined and blue: Unvisted Link
  • A visited link is underlined and purple: Visted Link
  • An active link is underlined and red: Active Link

Tips

Tip: If the <a> has no href attribute, it is only a placeholder for a link.

Tip: A linked page is normally displayed in the current browser window, unless you specify another target.

  • No target: Will open target page in current tab.
  • _blank: Will open target page in a new tab.
  • _parent: Will open target page in the parent iframe.
  • _self: Will open target page in the same iframe in which it was clicked.
  • _top: Will open target page in the topmost iframe.

Examples

Example

How to open a link in a new browser tab using the target _blank:

<a href="https://www.unr.edu/" target="_blank">Visit UNR's Website!</a>


Result: Visit UNR's Website!


Example

How to use an image from the web as a clickable link:

<a href="https://www.unr.edu/" target="_blank">
<img src="Link to desired image" alt="UNR Logo">
</a>


Result: Click the image below to be taken to UNR's Website!


Example

How to link to another place on the same page:

<a href="#Details">Go to the Details Section!</a>

<section id="Details">
<h3>Details</h3>
<p> ...
</p>
</section>


Result: Go to Details Section!
